Description

I don't have any hesitate to show this project, this is one of my beautiful art. Previously it's embeded with Arduino UNO to control watering plant automatically. I classify the soil moisture into 3 section, which are dry, normal and wet.

Everyday the RTC (Real Time Clock) will check the watering plant schedule time. If it was the time, then the soil moisture sensor will measure how moisture the soil is. So, if the soil already dry (measured by soil moisture sensor) and happens between watering schedule, the water pump will flush on until the soil moisture sensor measure the soil already wet.

In my level up project section, I change the Arduino UNO with NodeMCU8266. All of functional system above are keep working, I change this microcontroller 'cause I know it will get more some advantages. With this microcontroller I could connecting the device to Access Point (via WiFi). Again, I connecting the mobile phone with my custom android application for monitoring and controlling the device via the same Wifi that connected. In modern way we talking about IoT (Internet of Things) and here we go, this is one of IoT prototype. If you get attracted to try my project, don't procrastinate it. You could check my github up. Cheers ^-^
